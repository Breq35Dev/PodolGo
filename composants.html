Composants du PodolGo

Explorez les éléments techniques qui constituent notre podomètre

Vue d'ensemble des composants

ESP32

Le cerveau du PodolGo est un microcontrôleur ESP32 à faible consommation d'énergie.

  • Fréquence : 240 MHz
  • Mémoire flash : 448 Kibits
  • Volt : 3.3V
  • Consommation : <78,32 mW

LSM6DS3 - Grove 6-Axis Accelerometer & Gyroscope

Un accéléromètre 3 axes qui comptes les pas.

  • Plage de mesure : ±2g, ±4g, ±8g, ±16g
  • Compatibilité : Arduino / Raspberry Pi
  • Consommation : 1.25 mA en mode actif
  • Interface : I2C

Grove - OLED Display 0.96" (SSD1315)

L'écran OLED monochrome de 0.96 pouces offre une excellente lisibilité même en plein soleil. Sa résolution de 128x64 pixels permet d'afficher clairement toutes les informations utiles.

  • Taille : 0.96 pouces
  • Résolution : 128x64 pixels
  • Contrôleur : SSD1315
  • Interface : I2C
  • Consommation : 20mA 3.3V / 5V en fonctionnement

Alimentation

Le PodolGo est alimenté par une pile bouton CR2032 standard, facilement remplaçable. Un circuit de gestion d'alimentation optimise la consommation pour maximiser l'autonomie.

  • Type de pile : CR2032 (3V)
  • Capacité : 220mAh
  • Autonomie : Variable selon l'utilisation
  • Consommation moyenne : <0.5mA

Schéma de fonctionnement

Le circuit électronique du PodolGo est conçu pour optimiser l'efficacité énergétique tout en assurant des performances fiables. Voici le fonctionnement global du système :

  1. L'accéléromètre détecte les mouvements et envoie les données au microcontrôleur
  2. Le microcontrôleur traite ces données à l'aide d'algorithmes avancés pour déterminer s'il s'agit d'un pas
  3. Le compteur de pas est incrémenté et les calories sont calculées en fonction de paramètres préétablis
  4. Ces informations sont affichées sur l'écran OLED en temps réel
  5. Le système entre en mode basse consommation entre les mesures pour économiser l'énergie
